Dies ist eine sehr einfach gehaltene Angular Applikation, die es ermöglicht, Abstimmungen über die Ethereum Blockchain mithilfe von SmartContracts zu tätigen.

Funktionsweise:

Abstimmen: Auf der Seite "Vote" für oder gegen Initiativen Stimmen.

Resultate: Auf der Seite "Results" können die bereits abgegebenen Stimmen eingesehen werden.

Abgrenzungen:

Da diese kleine Applikation lediglich als Proof-of-Concept dient, wurde auf ein Login verzichtet.
Ebenfalls kann momentan ein Benutzer mehrmals für die gleiche Initiative abstimmen, dies ist aus Gründen der besseren Testbarkeit so im SmartContract implementiert.
